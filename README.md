## ApiSetup CMS & API


#### Setup
Install postgresql
```
After install, should modify postgresql timezone.
```

Install fresh
```
go get github.com/pilu/fresh
```

Install dependencies
```
dep ensure
```

Database
```
store:store@localhost/store
```


#### Run
Run project:
```
fresh
```
